Logging of data

When you visit this website, data is stored on servers of the Augsburg University of Applied Sciences for security purposes, which may allow identification:

IP address of the requesting computer
Date and time of access
Name and URL of the retrieved file
Message as to whether the retrieval was successful
Operating system and browser of the requesting computer

The temporary storage of the IP address by the system is necessary to enable delivery of the website to the user's computer.
This data is stored for a maximum of seven days. The storage takes place for reasons of data security in order to guarantee the stability and operational safety of the system. There is no personal utilisation. The data will not be passed on to third parties. No personal data is transferred abroad. A statistical evaluation of the data does NOT take place, not even in anonymous form.

Cookies and JavaScript

Cookies and JavaScript applications are only used here for session management and extended navigation options. The use of this functionality can be deactivated by the respective user through settings in the browser.
